# 🗣️ Piktos Portal v0.2.1

> „Hlas pro ty, kteří ho hledají."

Komunikační aplikace pro děti s autismem. Postaveno s láskou pro Jiříka a Štěpánka.

## Rychlý start

```bash
npm install
npm run dev
```

## Produkční build

```bash
npm run build
```

## Technologie
React 18 · TypeScript · Vite · Tailwind CSS · ARASAAC API
